## Description ##

An include featuring a complete wrapper of the Discord Webhook API written in SourcePawn using methodmaps and the [RIPExt extension](https://github.com/ErikMinekus/sm-ripext).

## Requirements ##
- Sourcemod and Metamod
- RIPExt extension
